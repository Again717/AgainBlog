## 🚀 现在请按以下步骤操作

### 1. 停止当前运行的服务器（如果有）
按 `Ctrl+C` 停止当前进程

### 2. 重新启动服务器

**方式 A：使用 nodemon（推荐，自动重启）**
```bash
npm run dev
```

**方式 B：使用 node（普通启动）**
```bash
npm start
```

### 3. 验证服务器运行

启动成功后，你应该看到：
```
========================================
MongoDB 连接成功
服务器运行在 http://localhost:3000
Travel Blog API 已启动
========================================
```

### 4. 测试 API

在浏览器访问：http://localhost:3000

应该看到：`{"message":"Travel Blog API is running!"}`
